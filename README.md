## Installation Instructions

### Client

1. run 'cd client'
2. run 'npm install'
3. run 'npm run dev'

### Server

1. Open s sperate terminal window
2. run 'cd server'
3. Create .env file and include REACT_APP_bearer_token=XXX'
4. run 'npm install'
5. run 'cd back-end'
6. run 'npm run start'
